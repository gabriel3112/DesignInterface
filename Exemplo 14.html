Exemplo 14 HTML - Formularios

Os formularios HTML são uma das principais formas de se interagir com aplicações web. Esses formularios utilizam diversos componentes que melhorarão a experiência do usuario

Para trabalhar com formularios devemos usar a tag <form> em conjunto com outras tags que vão representar os componentes de formularios, tais como: labels(rotulos), inputs(caixa de texto), botões, elementos de checagem, de seleção, entre outros componentes. Neste exemplo, demonstraremos o uso dos métodos Get e Post

Métodos do Protocolo HTTP

Get

Quando trabalhamos com a criação de formularios HTML, nós deveremos na maior parte das vezes utilizar o protocolo de rede HTTP, esse protocolo possui varios metodos que são utilizados para facilitar o processo de comunicação entre o cliente e o servidor

o método GET é aquele que anexa os dados do formulario à uma URL, esses dados do formulario são visiveis no endereço pu na barra de endereço do navegador.

è importante saber que este método não é recomendado para operações de inclusão, pois os dados vão aparecer para qualquer pessoa, o que pode ser inseguro.

Post

Já o metodo POST é aquele que envia os dados para o servidor no corpo da página. Essas informações não são visiveis para o cliente, sendo um pouco mais seguras que quando utilizamos o metodo GET
