Exemplo 9 - HTML

Exemplo sobre Web Semântica

<header>

Este bloco refere-se a tag semântica <header> que representa o cabeçalho de uma página. Nele geralmente utilizaremos as tags de seção/título

<nav>

Neste bloco representamos a seção de navegação que utiliza a tag <nav>. Neste bloco definimos por exemplo os menus de navegação do nosso documento html. Por exemplo:

  • Início
  • Cadastro
  • Relátorios
  • Movimentação
  • Sobre

<main>

A tag <main> utilizada neste bloco representa o conteúdo principal da página. Nele teremos as tags semânticas <section>, <article> e <aside>.

<section>

Neste bloco estamos usando a tag semântica <section> para definir uma seção genérica de um documento.

HTML - Web Semântica

A web semântica refere-se ao uso do HTML e outras tecnologias para estruturar o conteúdo de uma página de forma que ela seja compreendida tanto por humanos quanto por máquinas. Isso significa usar elementos HTML que descrevam o significado do conteúdo em vez de apenas sua aparência.

<article>

A tag semântica <article> define um conteúdo independente , como uma postagem de bloco ou notícia.

<aside>

A tag semântica ><aside> representa um conteúdo auxiliar, relacionado mais não essencial, ele é utilizado como um menu auxiliar.

  • menu auxiliar 1
  • menu auxiliar 2
  • menu auxiliar 3
  • menu auxiliar 4

<footer>

A tag semântica <footer> é aquela que define um rodapé do documento HTML. Nela geralmente colocamos informações de fechamento do nosso documento, tais como nome, autor, data, etc.
