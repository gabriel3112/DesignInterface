Visão Geral sobre CSS

CSS é um acronimo para folhas de estilo em cascata. Essa tecnica visa personalizar, estilizar um documento HTML. Para isso usamos regras CSS, que representa uma estrutura que define como os elementos selecionados devem ser estilizados. Essa regra é composta pelas seguintes partes:

  • Seletor - Indica quais elementos serão afetados pelo estilo definido.
  • Bloco de Declaração - É o bloco composto por propriedades e valores, esse bloco será aplicado ao elemento HTML.

Tipos de seletores

Seletores de ID

O seletor de ID é aquele que aplica a regra CSS a um componente HTML que tenha um atributo ID no componente HTML. Este Seletor de ID deve ser aplicado a um unico elemento HTML, haja vista que o ID deve ser único. Este seletor tem prioridade sobre os demais seletores (de classe e de tipo).

Este seletor sempre começara com um "#", seguido pelo nome do seletor e suas propriedades e valores.
