Visão Geral sobre CSS

CSS é um acronimo para folhas de estilo em cascata. Essa tecnica visa personalizar, estilizar um documento HTML. Para isso usamos regras CSS, que representa uma estrutura que define como os elementos selecionados devem ser estilizados. Essa regra é composta pelas seguintes partes:

  • Seletor Indica quais elementos serão afetados pelo estilo definido.
  • Bloco de Declaração è o bloco composto por propriedades e valores, esse bloco será aplicado ao elemento HTML.
