exemplo 4 html - Estrutura Geral de um Doc HTML

Este exemplo mostra a estrutura minimade um doc html. Aqui estão as principais tags utilizadas:

  • <!DOCTYPE html >: Define o tipo de doc HTML, ou seja a versão do html que será usada na págima.
  • <html >: A tag hmtl é o elemneto principal do documento html, ela define o contâiner raiz de toda págima
  • <head >: A tag head contém os metadados que serão interpretados pelo navegador. Nela encontraremos o tipo da página e links e arquivos para configuração do doc.
  • <body >: A tag body é responsável pela exibição do conteúdo vísivel do doc, ou seja, tudo que estiver neste bloco será vísivel para o usuário.
