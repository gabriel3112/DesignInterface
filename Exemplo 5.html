Exemplo 5 HTML - Tags de Seção e Titulos

Este exemplo mostra como usar as tags de titulo <h1> à <h6> para organizar o conteúdo do doc html em seções hierarquicas.

Descrição

As tags de titulo são usadas para definir cabeçalhos e subtitulos. Elas variam de <h1> (mais importante) à <h6> (menos importante).
